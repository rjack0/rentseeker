# Stack Grounding

This desktop dashboard is being built against official projects and documentation as a build-time aid, not as a runtime dependency.

## Local reference clones

- `.references/duckdb`
- `.references/electron-quick-start`
- `.references/glide-data-grid`
- `.references/maplibre-gl-js`

## Official docs consulted

- Electron main/renderer IPC and utility process docs
- DuckDB Node Neo and SQL extension docs
- MapLibre GL JS docs and protocol API docs
- PMTiles MapLibre integration docs
- deck.gl interactivity and map overlay docs
- Glide Data Grid quick-start and API docs

## Why this exists

The goal is to keep the Electron dashboard aligned with real extension points and documented APIs instead of fabricated wrappers.
